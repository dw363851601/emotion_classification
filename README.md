## Clone repository
```
$ git clone https://github.com/s4lat/emotion_classification

```
## If you on raspberry execute install_rasp.sh
```
$ ./install_rasp.sh
```

## If you on PC, install python3 modules from requirements.txt
```
$ python3 -m pip install -r requirements.txt
```

## Run main.py
```
$ python3 main.py
```
